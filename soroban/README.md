# soroban

そろばんの運珠で時刻を告げるそろばん時計

HTML5 Canvas APIとVanilla JS（純粋なJavaScript）のみで構築された、日本の伝統的な計算器「そろばん」をモチーフにしたデジタル時計です。外部ライブラリを一切使用せず、軽量かつ高速な描画（60FPS）と、和モダンで風情のある立体的なデザインを実現しています。

## 特徴

- **6桁そろばんレイアウト**: 時（2桁）、分（2桁）、秒（2桁）の計6桁で時間を表現します。時・分・秒のグループ間に太い仕切り（セパレーター）と十分な隙間を設け、デジタル時計としての可読性と、そろばんとしての伝統的なたたずまいを両立させています。
- **3Dライクな立体表現**: そろばん玉特有の「上下に鋭角なひし形」の珠、木目の美しい外枠、軸（芯棒）、定位点（梁の上の黒いポッチ）を、Canvasのグラデーションとシャドウを重ねて超高精細に描画。プラスチックや木材、象牙のような上質な質感を表現しています。
- **イージング付き運珠アニメーション**: 時刻が変化する際、珠がパチッと上下に滑らかにスライドするイージングアニメーションを実装。珠同士が連動して動くそろばんらしい物理的な挙動を再現しています。
- **3つの美しい和モダンテーマ**:
  - **欅琥珀 (Default)**: 欅（けやき）調の美しい赤みの木枠に、琥珀色とブロンズの珠を合わせたクラシックな配色。
  - **漆黒 (Shikkoku)**: 漆塗りの黒いフレームに、朱赤のアクセント、深い緑檀（りょくだん）調の緑の珠を合わせた伝統的な配色。
  - **白磁 (Hakuji)**: 現代的で静かな青磁色のフレームに、乳白色・象牙調の珠を合わせたモダンで清潔感のある配色。
- **Widget鑑賞モード**: 右上のギアアイコンをクリックすることで、設定パネルを完全に非表示にし、そろばんが静かに時を刻む様子を純粋に鑑賞できます。

## 使い方

### 1. ローカル環境で実行する
プロジェクトフォルダ内の `index.html` をダブルクリックし、Google Chrome、Safari、Firefox、Edgeなどの任意のウェブブラウザで開くだけで即座に動作します。インストール作業などは不要です。

### 2. コントロールパネルの操作方法
- **ギア（歯車）ボタン**: 画面右上のギアボタンをクリックすると、設定パネルの表示と非表示（Widget鑑賞モード）を切り替えられます。
- **色彩テーマ**: 3つの色彩テーマ（欅琥珀、漆黒、白磁）を切り替えられます。
- **動作モード**: 「現在時刻同期（実際の時間に合わせて運珠）」と、24時間がおよそ3時間で一周する「高速デモ」モードを切り替えられます。

## 技術仕様

- **描画エンジン**: HTML5 Canvas 2D Context (2DグラフィックスAPI)
- **ロジック**: Vanilla JS (ES6+)
- **スタイリング**: Vanilla CSS (Glassmorphism & Wa-Modern Design)
- **Webフォント**: Google Fonts (Outfit, Shippori Mincho, Share Tech Mono)
