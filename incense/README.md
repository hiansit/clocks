# incense

お香が燃え進んで時刻を告げる香時計

HTML5 Canvas APIとVanilla JS（純粋なJavaScript）のみで構築された、日本の伝統的な「香盤時計（香時計）」をモチーフにした、視覚的に美しい物理シミュレーションデジタル時計です。外部ライブラリを一切使用せず、軽量かつ高速な描画（60FPS）と、和モダンで風情のあるデザインを実現しています。

## 特徴

- **一筆書き24時間連結お香経路**: 6列×4行（24時間）のグリッド状にお香の経路を配置し、偶数行と奇数行で進行方向を左右反転（S字蛇行）して一筆書きで連結する精巧な数理座標モデルを搭載しています。
- **3つの美しい和風テーマ**: 漆黒（漆と朱の対比）、白磁（透明感あるグレーと萌黄色）、紫檀（重厚な木目調と金泥のお香）の3つの配色を瞬時に切り替え可能です。
- **リアルな燃焼・煙エフェクト**: 炭火のように明滅する火点（赤熱グロー）と、そこから立ち上りゆらゆらと広がる半透明の煙パーティクルによる物理シミュレーションを実装しています。
- **伝統とモダンを融合した目盛り**: 0〜23時の時間表示には、対応する十二支（「子初」「子正」など）を併記。1ユニット内（1時間分）のジグザグ経路には10分〜50分のドット目盛りと、30分の位置に「半」の文字を美しく配して時刻の可読性を高めています。
- **Widget鑑賞モード**: 右上のギアアイコンをクリックすることで、設定パネルを完全に非表示にし、お香が燃えて煙がたなびく様子だけを純粋に鑑賞できます。
- **静的ファイル構成**: HTML・CSS・JavaScriptが1つのファイル（`index.html`）に凝縮されているため、サーバーへの配置や配布が極めて容易です。

## 使い方

### 1. ローカル環境で実行する
プロジェクトフォルダ内の `index.html` をダブルクリックし、Google Chrome、Safari、Firefox、Edgeなどの任意のウェブブラウザで開くだけで即座に動作します。インストール作業などは不要です。

### 2. コントロールパネルの操作方法
- **ギア（歯車）ボタン**: 画面右上のギアボタンをクリックすると、設定パネルの表示と非表示（Widget鑑賞モード）を切り替えられます。
- **色彩テーマ**: 3つの色彩テーマ（漆黒、白磁、紫檀）を切り替えられます。背景のグラデーション、お香、灰、火点の色がシームレスに変化します。
- **動作モード**: 「現在時刻同期（実際の時間に合わせてゆっくり燃焼）」と、24時間がおよそ70分で一周するダイナミックな「高速デモ」モードを切り替えられます。
- **煙の発生頻度**: スライダーを動かして画面上に立ち上る煙の量を調整できます（無〜多）。

## 技術仕様

- **描画エンジン**: HTML5 Canvas 2D Context (2DグラフィックスAPI)
- **ロジック**: Vanilla JS (ES6+)
- **スタイリング**: Vanilla CSS (Glassmorphism & Wa-Modern Design)
- **Webフォント**: Google Fonts (Outfit, Shippori Mincho, Share Tech Mono)
