# sandclock

砂が舞って文字になるデジタル時計

HTML5 Canvas APIとVanilla JS（純粋なJavaScript）のみで構築された、視覚的に美しい砂の物理シミュレーションデジタル時計です。外部ライブラリを一切使用せず、軽量かつ高速な描画（60FPS）を実現しています。

## 特徴

- **超高速な文字収束物理**: ターゲットシャッフルの廃止と直接吸着（スナップ）演算により、秒の切り替わりの瞬間にも一瞬（約0.1秒以内）で極めてシャープな文字が形成されます。
- **4つの美しいカラーテーマ**: Midnight Gold（黄金）、Rose Quartz（桜）、Aurora Teal（極光）、Neon Pulse（サイバーパンク）を瞬時に切り替え可能です。
- **レスポンシブ設計**: PCの大画面からスマートフォンの縦長画面まで、ブラウザの画面サイズに合わせて時計のスケールが自動調整されます。
- **Widget鑑賞モード**: 右上のギアアイコンをクリックすることで、設定パネルを完全に非表示にし、時計の演出だけを純粋に鑑賞できます。
- **インタラクティブな設定項目**: 砂の総数や砂の大きさをスライダーでリアルタイムに変更可能です。
- **静的ファイル構成**: HTML・CSS・JavaScriptが1つのファイル（`index.html`）に凝縮されているため、サーバーへの配置や配布が極めて容易です。

## 使い方

### 1. ローカル環境で実行する
プロジェクトフォルダ内の `index.html` をダブルクリックし、Google Chrome、Safari、Firefox、Edgeなどの任意のウェブブラウザで開くだけで即座に動作します。インストール作業などは不要です。

### 2. コントロールパネルの操作方法
- 画面右上の **ギア（歯車）ボタン** をクリックすると、設定パネルの表示と非表示（Widget鑑賞モード）を切り替えられます。
- **テーマ選択**: 4つのカラー丸ボタンから、お好みのテーマに切り替えられます。背景のグラデーションと砂の配色が調和して変化します。
- **時間フォーマット**: 「HH:MM:SS（時：分：秒）」表示と「HH:MM（時：分）」表示を切り替えられます。
- **砂の総数**: スライダーを動かして画面上の砂粒の量を変更できます（1000〜8000粒）。※端末の描画スペックに合わせて調整してください。
- **砂の粒径**: 砂粒の大きさを変更できます（1.0px〜3.0px）。

### 3. GitHub Pagesで世界中に公開する
このプロジェクトは単一の静的ファイルのみで構成されているため、GitHub Pagesを使って無料で簡単に公開できます。
1. 本リポジトリをご自身のGitHubアカウントにプッシュします。
2. GitHubの対象リポジトリの **Settings**（設定）を開きます。
3. 左メニューの **Pages** タブを選択します。
4. **Build and deployment** の Source で `Deploy from a branch` を選択し、公開するブランチ（`main` または `master`）とフォルダ（`/ (root)`）を指定して **Save** をクリックします。
5. 数分後、提示される専用URLにアクセスすると全世界に向けて公開されます。

## 技術仕様

- **描画エンジン**: HTML5 Canvas 2D Context (2DグラフィックスAPI)
- **ロジック**: Vanilla JS (ES6+)
- **スタイリング**: Vanilla CSS (Glassmorphism / ガラスモーフィズム設計)
- **Webフォント**: Google Fonts (Outfit, Share Tech Mono)
