# Clocks Collection - 変わり種時計コレクション

ユニークなビジュアルと物理演算を組み合わせた、ブラウザ上で動作する「変わり種デジタル時計」の統合コレクションリポジトリです。

すべての時計は外部ライブラリを使用せず、HTML5 Canvas 2D APIと純粋なCSS/JavaScript（Vanilla JS）のみで軽量かつ高速に動作するよう設計されています。

## 時計コレクション一覧

現在、以下のユニークな時計が公開されています。

### 1. [砂の時計 (Sand Clock)](./sandclock/)
- **概要**: 時刻を表す文字が、風と重力に影響を受ける数千粒の「砂」によって有機的に形成・崩壊・再集結するデジタル時計です。
- **特徴**: ターゲット座標への超高速な吸着物理演算により、秒表示でも一瞬で極めてシャープな文字が浮かび上がります。Midnight GoldやRose Quartzなど4つの美しいカラーテーマや、砂の量・サイズの設定機能を搭載しています。

## 動作方法

### ローカルでの動作
ルート直下にある `index.html`（ポータル画面）をダブルクリックし、任意のウェブブラウザで開くことで、すべての時計アプリへ簡単にアクセスして起動することができます。

### GitHub Pagesでの公開
1. 本リポジトリをご自身のGitHubアカウントにプッシュ（またはフォーク）します。
2. リポジトリ設定（Settings）の **Pages** から、公開するブランチ（`main` など）とフォルダ（`/ (root)`）を指定して保存します。
3. 数分後、自動的にポータル画面とすべての時計アプリがWeb上に公開されます。

## 技術仕様

- **レンダリング**: HTML5 Canvas 2D Context
- **ロジック**: Vanilla JS (ES6+)
- **デザイン**: CSS Grid / Flexbox / Glassmorphism
